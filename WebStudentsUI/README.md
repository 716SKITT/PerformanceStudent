# WebStudentsUI

Frontend-приложение Angular для проекта `PerformanceStudent`.

## Быстрый старт (локально)

```bash
npm ci
npm run start
```

По умолчанию dev-сервер доступен на `http://localhost:4200`.

## Сборка

```bash
npm run build
```

Артефакты: `dist/webstudents-ui`.

## В Docker

В основном сценарии frontend запускается через `docker compose` из корня репозитория.
